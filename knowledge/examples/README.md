# Research Examples

This directory contains real-world examples of research questions and recommended approaches.

## Purpose

Examples serve multiple purposes:
- **Few-shot learning**: Help the LLM understand expected output format
- **User guidance**: Show users what kinds of questions work well
- **Quality assurance**: Provide test cases for validating recommendations
- **Training**: Build institutional knowledge about effective research design

## Structure

Examples are organized by category:
- `pricing-examples.json`: Pricing and monetization research
- `cx-examples.json`: Customer experience and journey research
- More categories to be added (features, brand, competitive, etc.)

## Format

Each example includes:
- The original question
- Context about the business situation
- Recommended research approach
- Key insights or rationale

## Usage

These examples can be:
- Loaded into LLM prompts as few-shot examples
- Displayed to users for inspiration
- Used in test suites
- Referenced in documentation
