# QuestionPro Context Data

This directory contains processed knowledge from the QuestionPro workspace that informs the AI Research Guide's recommendations.

## Files in this Directory

### pricing-summary.md
Contains QuestionPro pricing information, tier structures, and feature availability by plan.

### cx-capabilities.md
Details about QuestionPro's CX Reputation platform and customer experience features.

### methodologies-guide.md
Comprehensive guide to executing research methodologies using QuestionPro's platform.

### platform-outputs.md
Describes the outputs, reports, and deliverables available from QuestionPro.

## Purpose

These files serve as reference material to:
1. Enhance LLM recommendations with accurate QuestionPro-specific details
2. Ensure implementation guidance matches actual platform capabilities
3. Provide realistic output expectations
4. Support accurate pricing and tier information in responses

## Maintenance

These files should be updated when:
- QuestionPro releases new features
- Pricing or packaging changes
- New methodologies or capabilities are added
- Best practices evolve

## Integration

The main application can optionally load these files to:
- Inject context into LLM prompts for more accurate responses
- Validate recommendations against actual platform capabilities
- Provide detailed implementation guidance
- Support advanced features like pricing guidance
